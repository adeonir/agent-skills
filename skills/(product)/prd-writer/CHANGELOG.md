# Changelog

All notable changes to this skill will be documented in this file.

## v1.0.0 (2026-02-08)

### Added

- **Skills Format Migration**: Created as new skill from design-builder plugin
- **Sources**:
  - `agents/product-planner.md`: Product definition content (value proposition, audience, features, structure)
  - Awesome Copilot PRD patterns: 3-phase workflow, quality standards, schema structure
- **3-Phase Workflow**: Discovery interview, Analysis & Scoping, Technical Drafting
- **Hybrid PRD Schema**: Executive Summary, Product Definition, User Experience, Technical Specs, Risks & Roadmap
- **Quality Standards**: Concrete, measurable requirements (no vague adjectives)
- **Integration Points**: Cross-references with design-builder and spec-driven skills
- **Output Format**: PRD written to `.specs/docs/prd-{project-name}.md`
- **Versioned Roadmap**: Uses versioned rollout (v1.0, v1.1, v2.0) instead of MVP framing
